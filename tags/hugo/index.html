Get up and running with a free Website in 15 Mins

A one page instructable Basic Setup: 5 minutes Create a free github account say demouser. This repository will contain the rendered version of the website. Use the github dashboard to create a public github repository with the name demouser.github.io Install hugo on the local machine. On MacOS: brew install hugo Create a new site, say named mynotes hugo new site mynotes Choose a theme from available themes The recommended theme is good to get started with.
