Get up and running with a free Website in 15 Mins

August 23, 2018

A one page instructable

Basic Setup: 5 minutes

  • Create a free github account say demouser. This repository will contain the rendered version of your website.
  • Use the github dashboard to create a public github repository with name demouser.github.io
  • Install hugo on your local machine. On MacOS:
   brew install hugo
  • Create a new site, say named mynotes
  hugo new site mynotes

  • Choose a theme from available themes The recommended theme is good to get started with. Get a feel of the theme at the demo page .

    cd quickstart
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

    title = "My Notes"
baseURL = "http://demouser.github.io/"
theme = "ananke"

  • Add content to play around with features and look.

  hugo new posts/sandbox.md
  • Start local server to preview. -D allows serving of draft content.
  hugo server -D

Open your browser to http://localhost:1313/ to preview the site.

  • For basic customization of background image, font etc. Modify configuration in mynotes/config.toml. See notes/themes/ananke/README.md and refer to notes/themes/ananke/exampleSite.

  • When mildly satisfied with the site prepare to host it publicly. Remove draft status from file. draft: false in config.toml or use the command

    hugo undraft

    Ctrl-C to kill the hugo server. Then remove the generated files in preparation of setting up the remote repository from a clean slate.

    rm -rf public

    Setting the repository as a submodule, allows content to be stored separately from generated files used to render the website.

    git submodule add -b master git@github.com:demouser/demouser.github.io.git public
hugo
cd public
git commit -m "checking liveness"
git push origin master

    Optional Setup: 5 minutes

  • Further customization. Create _index.md files ( see notes/themes/ananke/exampleSite ) to further tweak appearance.

  • Add google analytics. Add your website http://demouser.github.io as a new property in http://analytics.google.com/ You will get a tag like UA-1234567-1. Add it to config.toml add your

  googleAnalytics = "UA-12345678-1
  • Enable comments on your posts. Create an account at disqus Add the following to config.toml
  disqusShortname = "http-demouser-github-io"
  • Contact form to relay email. Create mynotes/content/posts/contact.md

{{< form-contact action=“https://formspree.io/yourusername@domain.com" >}}

  • Store content in different git repository. Create a repository named mynotes somewhere. It need not be in github.
  cd mynotes
  git rm public --cached
  git add .
  git commit -m "getting started with website content"
  git remote add origin git@bitbucket.org:usernameonbitbucket/mynotes.git
  git push -u origin master

Write Documentation: 5 minutes

  • Install markdown-mode for emacs from melpa. C-c c-h to see all keybindings.
  • Document all your steps for future reference.
